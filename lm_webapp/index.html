App server mode for LM

The LM system combines DSP (main LM unit ) ESP for connectivity (WIFI or Bluetooth) or WT (Classic Bluetoot Module) A client (Browser running on PC or Android) with a webapplication The WebApplication can be served in different ways

WIFI Use WIFI and Websocket
Both ESP and Client needs to be on the same WIFI network
The IP of the ESP should be:192.168.123.87

BLUETOOTH Use Classic Bluetooth
Rememeber to setup the ESP:
  • switch on Bluetooth by issuing a M3 command
  • swithc off WIFI by M2
    This can be done by connecting the ESP connected via USB cable select a serial connection in the consolen in the console

    • LOCALHOST For development purpose, serving from a localhost (xampp)
    Both ESP and Client needs to be on the same WIFI network
    The IP of the ESP should be:192.168.123.87
